# Homebrew Tap

Homebrew tap for Symaira and OpenPass.

## Installation

```bash
brew tap danieljustus/tap
```

## Available Formulae & Casks

| Name | Type | Install |
|---|---|---|
| `symvault` | Formula | `brew install symvault` |
| `symaira-terminal` | Cask | `brew install --cask symaira-terminal` |
